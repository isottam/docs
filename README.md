# isottam/docs

Public documents published in Matthew's house style.

Each subfolder is one document. Open any document at
`https://isottam.github.io/docs/<slug>/`.

**This repository is regenerated** from a private toolchain — do not hand-edit
the HTML. Authoring happens in YAML; the renderer produces the per-document
folders and rebuilds the index.

GitHub Pages serves from `main` / root.
